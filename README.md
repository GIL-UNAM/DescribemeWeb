# DescribemeWeb

Este es el repositorio del sitio web de Descríbeme.

### Servidor de desarrollo

Para iniciar el servidor de desarrollo con hot-reload, ejecuta el siguiente comando. El servidor estará disponible en http://localhost:3000:

```bash
pnpm run dev
```

### Compilación para producción

Para compilar el proyecto para producción, utiliza:

```bash
pnpm run build
```
